# Onboarding by role

Pick your role. Each guide is a 5-day plan covering tools, repo orientation,
your first PR, and what "done" looks like by day 5.

| Role | Guide |
|---|---|
| Data Engineer | [data-engineer.md](data-engineer.md) |
| Data Analyst | [data-analyst.md](data-analyst.md) |
| Data Scientist | [data-scientist.md](data-scientist.md) |
| Business Analyst | [business-analyst.md](business-analyst.md) |

If your role isn't listed, start with `data-engineer.md` and skip steps that
don't apply — every contributor needs at least the tools and the
`AGENTS.md` reading list.
